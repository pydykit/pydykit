# About `pydykit`

## Development

### Start on new machine

```bash
python3.12 -m venv .venv
source .venv/bin/activate
pip install -r requirements_dev.txt
```

### Install Python package `pymetis` in editable-/develoment-mode

```bash
pip install --editable .
```
